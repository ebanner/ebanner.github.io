# SLIME

Start common lisp mode

```emac-lisp
SPC SPC common-lisp-mode
```

Start SLIME repl

```emac-lisp
SPC m s i ; slime
```

Eval last sexp

```emac-lisp
C-x C-e ; slime-eval-last-expression
```

Eval expression in repl

```emac-lisp
SPC m s e ; slime-eval-last-expression-in-repl
```
