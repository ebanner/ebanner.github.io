# Binary files

Viewing a binary file

```
hexdump -C file.bin
```

Viewing the contents of a binary file

```
xxd file.bin
```

Viewing how many bytes are in the file

```
wc -c file.bin
```

Viewing how many bytes are in the file

```
ls -lh file.bin
```
